# imaginary me! features

This is a web based application that has only a few basic ideas:

1. survey the user for their interests.  Favorite movies, books, tv shows.  Superpower you wish you had.  Places you want to go.

2. Allow the user to upload several profile pictures of themselves.

3. Ask for AI to take the combination of #1 and #2 and create several photos the user can download.  These photos should smartly integrate the items, creating unique combinations.

For example, if Dune was a favorite book or movie, one of the photos could be taking a profile pic and turning them into a species or character from the Dune universe.

The application should be simple to use.  It should be easy to change any parameter.

The AI provider and models to use are an app configuration item, not something the user has to worry about.

Not photos are ever kept.  the user can download them.  if they leave the page, the photos are gone.

The application should be mobile friendly

Bonus: if there are models or api's that also allow for sound or voice overs to be generated that also is based off of parameters from #1, that would be great to include as overlays or things to play while looking at the generated images.
